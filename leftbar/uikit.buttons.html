Buttons

Button size, color and shape variations.
Basic Examples

Simply add .btn to a button or a tag and use one of the coloring classes for example .btn-primary

Outline buttons

Instead of btn-primary use .btn-outline-primary

Rounded buttons

Add btn-rounded

Icons Buttons

For icon buttons use .btn .btn-icon and use one of the brand backgrounds like .bg-facebook

Flipping Icons Buttons

For flipping social icon buttons use.btn .btn-icon .btn-icon-flip and use the inner tag <i> twice. Inspect element to see the usage!

Disabled State

Make buttons look inactive by adding the disabled boolean attribute to any <button> element.
<a> s don’t support the disabled attribute, so you must add the .disabled class to make it visually appear disabled.

Sizes

Fancy larger or smaller buttons? Add .btn-lg or .btn-sm for additional sizes

Block Buttons

Create block level buttons—those that span the full width of a parent—by adding .btn-block.
