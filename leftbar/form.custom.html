Custom Form Elements

Form customized checkbox, radio elements.

Checkboxes

Basic Usage

Wrap <input type="checkbox"> within a <div class="checkbox"> For default usage

Without label text
Color Options

Add color class e.g. checkbox-primary

Size Options

Addcheckbox-lg for larger checkbox

Inline Checkbox

Add class checkbox-inline

Different Shapes

Add checkbox-circle for circular checkboxes and checkbox-square for squared corners

Awesome bootstrap checkbox Documentation

Radio Buttons

Basic usage

Wrap <input type="radio"> within a <div class="radio"> For default usage.

Color options

Add color class e.g. radio-primary

Inline radios

Add class radio-inline for inline radios

Without label text

Switches

Basic Usage

Add data-plugin="switchery" to a checkbox for deafault switch

Color Options

Add data-color="#xxx" to change color
and data-jack-color="#xxx" to change jack color

Size Options

Add data-size="small" or data-size="small" to change size
